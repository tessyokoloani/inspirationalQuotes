Prerequisites

    npm or yarn
    
Installation

    Clone the repository:

    git clone https://github.com/your-username/inspirational-quotes-app.git
    cd inspirational-quotes-app

    Install dependencies:

    npm install
    # or
    yarn install
    
    Start the development server:
    
    npm run dev
